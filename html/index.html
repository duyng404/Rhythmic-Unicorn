What is this about?

Rhythmic Unicorn is a crowdsourcing project that looks to build a music suggestion database through gamification. We, the team (Duy Nguyen & David Schabelski) wanted to experiment with an idea: an application that is built like a game, has the elements of a game and can used to create valuable output. When people play the game, they help building a database of related songs. This database can then be used by online music services like Google Play Music, Youtube Music or Spotify to help make better suggestions.

In the game, you will be given a random song, and then challenged to list out other songs that is related to it, as many as possible. The songs can be similar in genre, musical style, or simply the feelings, or "vibe", you have listening to it.

The more you answer, the more Contribution Tokens you gain. You can then spend these Tokens to request contribution for a specific song or to browse through the database itself.

The project is still a WIP, with a lot more stuff is coming in the future. Including a research paper, database statistics, and tons of updates and features!

Rhythmic Unicorn is built on Node.js, Expressjs and MongoDB. We also made use of Spotify API and Youtube API for its music database and for previewing songs.
